# Review - 코드 리뷰

현재 변경사항 또는 지정된 파일/기능에 대해 코드 리뷰를 수행합니다.

## 리뷰 항목

1. **코드 품질**
   - 가독성, 명명 규칙
   - 중복 코드
   - 함수/컴포넌트 크기

2. **버그 가능성**
   - null/undefined 처리
   - 에러 핸들링
   - 엣지 케이스

3. **성능**
   - 불필요한 리렌더링
   - 메모리 누수 가능성
   - 번들 사이즈 영향

4. **보안**
   - XSS 취약점
   - 민감 정보 노출
   - API 키 하드코딩

5. **MFA 아키텍처**
   - Module Federation 규칙 준수
   - 공유 의존성 관리
   - Host-Remote 통신

## 출력 형식
- 심각도: 🔴 Critical / 🟡 Warning / 🟢 Suggestion
- 파일:라인 위치
- 문제점 및 개선 제안

---

$ARGUMENTS: 리뷰할 파일 경로 또는 기능 설명. 없으면 `git diff` 기반으로 변경사항 리뷰.
